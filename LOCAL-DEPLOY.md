# 내 컴퓨터를 서버로 — 로컬 배포 가이드 (Windows)

깃허브 없이, 이 PC에서 직접 수집·번역·빌드하고 로컬 웹서버로 띄우는 방법입니다.
번역은 **무료(구글 번역, 키 불필요)** 로 동작합니다. (Anthropic/OpenAI 키가 있으면 더 좋은 번역으로 자동 전환)

## 0. 준비 — Node.js 설치 (1회)
1. https://nodejs.org 에서 **LTS** 버전 다운로드 → 설치(계속 Next).
2. 설치 후 확인: 시작 → "cmd" 실행 → `node -v` 입력 → `v20.x` 같은 게 나오면 OK.

> 별도 `npm install` 은 필요 없어요. 수집·빌드 스크립트는 Node 기본 기능만 씁니다.

## 1. 데이터 채우기 + 사이트 만들기
`local` 폴더의 **`update.bat`** 더블클릭.
- 최신 기사 수집 → 무료 번역(한국어) → 이미지 채우기 → `site\index.html` 생성.
- 처음엔 1~3분 걸릴 수 있어요. 창에 `done ...` 이 뜨면 완료.

## 2. 로컬 서버로 띄우기
`local` 폴더의 **`serve.bat`** 더블클릭.
- 브라우저에서 **http://localhost:8080** 접속 → 사이트가 보입니다.
- 같은 와이파이의 폰·다른 PC에서도 보려면: 이 PC의 IP로 `http://192.168.x.x:8080`
  (IP 확인: cmd에서 `ipconfig` → IPv4 주소)
- 이 창을 닫으면 서버가 꺼집니다. 계속 켜두면 계속 접속 가능.

## 3. 자동 갱신 (10분마다)
**`auto-update.bat`** 더블클릭 → 10분마다 자동으로 수집·번역·빌드를 반복합니다.
- 이 창을 켜두는 동안 사이트가 계속 최신으로 갱신돼요.
- `serve.bat` 창과 함께 둘 다 켜두면: 자동 갱신 + 상시 접속.
- 창을 닫으면 중지.

### 컴퓨터 켜질 때 자동 시작(선택)
항상 켜두고 싶으면 Windows **작업 스케줄러**에 `update.bat`을 10분 주기로 등록하면 돼요. 원하면 등록 방법도 안내할게요.

## 더 좋은 번역을 쓰고 싶다면(선택)
무료 번역으로 충분하지만, API 키가 있으면 자동으로 더 좋은 번역을 씁니다.
- cmd에서 키를 설정한 뒤 `update.bat`을 같은 창에서 실행:
  - `set ANTHROPIC_API_KEY=sk-ant-...` (Claude), 또는
  - `set OPENAI_API_KEY=sk-...` (GPT), 또는
  - `set GEMINI_API_KEY=...` (이미지 AI 생성)
- ※ ChatGPT 구독(월정액)은 코드에 연결할 수 없어요. 위 키들은 각 사이트(platform.openai.com 등)에서 발급하는 **API 키**이고 구독과 별개입니다.

## 문제 해결
- `node 명령을 찾을 수 없습니다`: Node 설치 후 cmd(또는 PC)를 새로 켜세요.
- 번역이 가끔 영어로: 무료 구글 번역이 잠깐 제한된 경우입니다. 다음 실행 때 다시 한국어로 채워져요.
- 이미지가 일부 빔: og:image가 없는 X(트위터) 글입니다. `GEMINI_API_KEY` 설정 시 AI 이미지로 채워집니다.
